# ☕ Maratona Java

Repositório criado para acompanhar meus estudos em **Java**, utilizando a **Maratona Java** como base, com foco em **fundamentos da linguagem**, **boas práticas** e **preparação para certificação Java**.

## 🎯 Objetivo
- Consolidar os fundamentos da linguagem Java
- Praticar lógica de programação
- Fixar conteúdos cobrados em certificações Java
- Manter um histórico organizado da evolução nos estudos
- Servir como material de consulta futura
